# Simple Typescript Template

This project is for quickly building applications using [NodeJS](https://nodejs.org/en/about/) and  [Typescript](https://www.typescriptlang.org/).

## Development

>We use node version 16+

In order to run the project and perform development, clone the repository:

```bash
git clone git@github.com:bromanla/typescript-template.git
```

Download the dependencies.

```bash
npm install
```

To run compile the Typescript:

```bash
npm run build
```

To run rebuild the Typescript:

```bash
npm run build:clean
```

To run the application:

```bash
npm start
```

To run the app in development mode where code changes will trigger a build and restart through [nodemon](https://www.npmjs.com/package/nodemon):

```bash
npm run start:watch
```
